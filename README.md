<h2>Sobre 📝</h2>

<p align="center">Happy é um site criado com o objetivo de conectar pessoas que desejam mudar a vida de várias crianças que vivem em orfanatos ou lares que acolhem crianças em situações de risco.<br>
O projeto possui uma proposta incrível e serve para lembrar-nos que existem diversas crianças próximas a nós que necessitam  de um carinho especial, algo que elas talvez não tenham conseguido durante a vida.<br>
O layout do site foi desenvolvido com muito carinho, contando com uma interface amigável e de fácil navegação, alcançando todos os públicos que possam ter interesse no projeto. 👊🏼</p>

<h2>Instalação 👨‍💻</h2>
Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:

- <b>[Git](https://git-scm.com)</b>
- <b>[Node.js](https://nodejs.org/en/)</b>

### E então:

```
- Clone o repositório:
$ git clone https://github.com/ggonpereira/happy-next-level-week3.git

- Depois disso acesse o diretório:
$ cd happy-next-level-week3

- Instale a dependência necessária:
$ npm install express

$ npm install nodemon

$ npm install hbs

$ npm install sqlite-async

- Por fim, digite:
$ npm start
```
